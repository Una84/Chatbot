# Cybersecurity Awareness Bot

## Description
A C# console chatbot that educates users about cybersecurity topics.

## Features
- Voice greeting (WAV playback)
- ASCII art interface
- User interaction with name personalization
- Cybersecurity responses
- Input validation
- Styled console UI

## Technologies Used
- C#
- .NET Console App
- System.Media for audio
- GitHub Actions (CI)

## How to Run
1. Open project in Visual Studio
2. Run the application
3. Ensure `greeting.wav` is in the Assets folder
